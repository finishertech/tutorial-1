Seguirá um caminho simples e tranquilo

A execução deste tutorial é focada na resolução do problema que estou propondo. E para ser mais direto ao ponto para explicar algo, você não me verá perder tempo na digitação de código. Eu usarei meu editor de textos preferido, o Vim, apenas para navegação. E o Vim geralmente será invocado pela saída do comando git difftool. Em tutoriais básicos como este, eu prefiro a abordagem de não ter trabalho digitando código. Daí eu posso apenas ler e executar o tutorial com comandos simples e rápidos no intuito de chegar a um final feliz!

Sendo assim, neste tutorial eu não vou explicar:

  • Como instalar ou usar um IDE como o Visual Studio Code, por exemplo. Como eu falei, neste tutorial eu estou mais preocupado em te explicar o processo e o código do que escrevê-lo.

  • A instalação de quaisquer ferramentas. Elas serão instaladas numa máquina virtual (VM) que você montará sozinho ou através do Vagrant utilizando o meu projeto devbox. Os procedimentos para isso são simples.

Resumindo, todos os passos que seguiremos neste tutorial nos exigirão apenas isto:

  • Uma linha de comando onde executaremos comandos bem simples e scripts Bash.

  • Conhecimentos prévios (nem rasos, nem profundos) em:

    • HTML 5, CSS 3 e Bootstrap 4.

    • JavaScript (ES6).

    • Git.

  • Um browser.
