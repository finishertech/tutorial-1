9. Utilizando o create-react-app

Abra um novo shell e execute os comandos a seguir:

2$ cd $PROJECTS_DIR
2$ echo $PROJECT
finishertech-website

Como eu disse no capítulo anterior, o prompt 2$ indica que os comandos acima devem ocorrer em um segundo shell. Para criá-lo através do tmux você pode usar a combinação de teclas Ctrl+b+c.

Crie o projeto através do comando yarn (que já está instalado):

2$ yarn create react-app $PROJECT && cd $_

A execução do comando acima adiciona o script create-react-app ao PATH. E se isso ocorrer com sucesso, também altera o diretório corrente para finishertech-website.

Na última geração deste tutorial a versão desse script era a seguinte:

2$ create-react-app --version
3.4.1
