5. Configurando o seu nome e email para repositórios Git

Por default, quando você for começar a utilizar o Git nos próximos passos, se você não configurá-lo no ambiente que criou, você vai acabar criando repositórios utilizando o meu nome e mail. Mas, você é você e não eu (apesar de estar me imitando aqui neste tutorial)! 😁

Sendo assim, precisamos informar o teu nome e email nas configurações do Git. Para fazer isso, execute o comando a seguir:

$ git-config
Seu nome: Fulano da Silva
Confirma "Fulano da Silva" (S/n)?
Seu email: fulanodasilva@gmail.com
Confirma "fulanodasilva@gmail.com" (S/n)?
O git foi configurado:
  email = fulanodasilva@gmail.com
  name = Fulano da Silva

Esse script irá te perguntar (e confirmar) seu nome e email, como na saída de exemplo acima. Em seguida, ele configurará teus dados no Git.
