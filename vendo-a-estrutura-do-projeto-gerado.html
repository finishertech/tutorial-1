10. Vendo a estrutura do projeto gerado

A estrutura inicial criada pelo comando acima, excluindo o diretório node_modules, é esta:

2$ tree -I node_modules
.
|-- package.json
|-- public
|   |-- favicon.ico
|   |-- index.html
|   |-- logo192.png
|   |-- logo512.png
|   |-- manifest.json
|   `-- robots.txt
|-- README.md
|-- src
|   |-- App.css
|   |-- App.js
|   |-- App.test.js
|   |-- index.css
|   |-- index.js
|   |-- logo.svg
|   |-- serviceWorker.js
|   `-- setupTests.js
`-- yarn.lock

2 directories, 17 files

Além disso, o React, após criar essa estrutura, também faz o primeiro commit em nosso nome, como exemplificado na saída do comando a seguir:

2$ git log
commit 59d01af7901c38054972e152a8575e6670be280e
(HEAD -> master)
Author: Paulo Jerônimo <paulojeronimo@gmail.com>
Date:   Fri Apr 10 19:52:00 2020 +0000

    Initialize project using Create React App

Para iniciar a aplicação gerada, basta executar o seguinte comando:

2$ BROWSER=none yarn start
Esse é outro comando que procuraremos manter em execução, durante o decorrer de todo o tutorial Então, para prosseguir com novos comandos, você precisará abrir outra linha de comando (identificada neste tutorial pelo prompt "3$ ")
