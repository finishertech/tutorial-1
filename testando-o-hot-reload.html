O que é o git-apply?

O git-apply é uma função Bash disponibilizada pelo script install quando o executamos nesse passo. Esse é o seu código:

git-apply() {
  git apply $TUT_DIR/patch/$1
  [ "$HIDE_DIFFTOOL" ] || git difftool
  while shift
  do
    [ ! "$1" ] || git add "$1"
  done
}
