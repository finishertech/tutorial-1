Mania: codar "in english"

Houve um tempo em que eu misturava muito a linguagem utilizada na produção de código. Hora eu escrevia programas em português, hora em inglês. Contudo, esse tempo passou. Eu vejo que, num mundo globalizado e open source como o que estamos vivendo hoje, não é mais "aceitável?" a escrita de programas em português. Portanto, o código que crio para a Finisher.Tech é todo em inglês, mesmo que, para o usuário final, os produtos que eu crio sejam apresentados, também, em português.

O que é o git-history?

O git-history é uma função Bash disponibilizada pelo script install quando o executamos nesse passo. Esse é o seu código:

git-history() { git log --oneline | cut -d' ' -f 2-; }
