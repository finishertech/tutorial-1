Um último commit

Antes de deixar nosso código público, vamos adicionar a Licença MIT e fazer um último commit para que você, ou qualquer outra pessoa, tenham liberdade para utilizá-lo sem nenhum problema e para que ele fique igual ao código que está publicado sob a tag v0.9.2-src.

Para isso, vamos executar os seguintes comandos:

$ tut-add LICENSE
$ git commit -m 'Added LICENSE'

Essa versão, em termos de código, tem poucas mudanças em relação a versão 0.9.1-src do site Finisher.Tech (publicada em 22 de Dezembro de 2019). Basicamente, este tutorial foi atualizado e ajustado para utilizar uma nova versão do create-react-app e isso provocou uma alteração no código gerado.
