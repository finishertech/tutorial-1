2. Instalando a dev-box

A dev-box é um projeto que construi para criar um ambiente Linux (Arch Linux ou Ubuntu, neste momento). Eu o utilizo para criar meus laboratórios ou tutoriais, como este.

Ela é open source e usa uma Licença MIT.

A dev-box contém um conjunto de scripts de provisionamento, escritos em Bash, que podem ser utilizados para instalar ferramentas de desenvolvimento (ou quaisquer).
