8. Colocando o projeto em execução

Para servir esse código ao nosso browser cliente, faremos uso de um servidor local:

$ yarn global add serve
$ serve -l 8545

Mantenha esse comando em execução pois iremos acessar essa versão até o final deste tutorial. Sendo assim, para prosseguir com novos comandos, você precisará abrir outras linhas de comando. Uma forma de fazer isso é através do tmux.

Para identificar o shell que estaremos utilizando, eu utilizarei um número que ficará anterior ao sinal "$". Ou seja, se os comandos estiverem sendo digitados no shell 2 o seu prompt será "2$ ".

Agora é só abrir um browser em http://localhost:8545 para ver o conteúdo. Esse será o site que iremos criar. Observe que ele, nessa versão, é um site que não contém nenhum código minificado ou obfuscado, é bem legível e utiliza o Bootstrap.

Esse é o "ponto de largada" para a construção do projeto que criaremos com o uso do React a partir de agora.
